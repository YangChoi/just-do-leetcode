### 📗 진행 방식
```
 - 한 주에 한 토픽을 정하여 매일매일 해당 토픽에 대한 Leet code 문제를 푼다. 
 - 문제 제출 시간은 다음날 오전 8시까지
 - 제출 연장은 최대 3일까지 가능하다.
 - 매주 토요일에 자신이 고른 문제 하나를 상대방이 이해할 수 있도록 설명을 한다.

```
